# GravLMS

Dev env. frontend
- Uses nodejs v22.21.0  (via nvm)
- Uses angular CLI v21.0

Dev env. backend
- spins up backend apache + php + mysql + phpmyadmin via docker

## Fist time use
- run the init_db in the html folder (backend)
- Run 'docker compose up -d' in terminal window
- Run 'ng serve'
- default login data: (admin@example.com/password)
